Current datasets on TSA-ML platform

Taiwan Social Change Survey (TSCS) is a longitutindal dataset containing survey data on different social topics such as employment, family, and social networks. The TSCS contains data from the 1980's from individuals and families across Taiwan. This asset is managed by the Center for Survey Research, Academia Sinica, Taiwan. More detail information is available here.

World Values Survey (WVS) has a collection of survey datasets of social and economic indicators of individuals. It contains deidentified data from the 1980s of different countries including those of East-Asia, America, and Europe. The TSA-ML platform currently contains data from Hong Kong, Macao, Taiwan, and China. A network of researchers from different countries have rigiously sampled the population and collected representative information of individuals and families. More detail information is available here.

Interconnected Data

Automated Crosswalk Analysis

Natural Language Processing (NLP) and AI has been used to automate the crosswalk analysis of target survey questions

Regional Data at Scale

Current data focuses on East-Asian region (predominately, Hong Kong, Macau, Taiwan and China). Other datasets from neighbouring regions are welcome. A list of public data assets are listed here. Other groups can contribute to this repository.

Machine Learning and AI

Different aspects of the system uses machine learning and AI, including Natural Language Processing (NLP), which integrates different survey data. The system leverage's graph databases which links data sources to allow analysis and pre-training. A Github of resources, assets, and tutorials can be access here.

Supported by Research

Based on previous research in AI, several key demographic targets from the meta-data of public datasets were screened. 700+ variables were manually screened from 70+ datasets which contained demographic and socioeconomic survey questions. Research can be access here.

Your contributions are welcome...

Others are welcome to contribute. Please email your enquires to us. Instructions to setup the graph database containing the survey data is available in the Github repository. Tutorials are also available. The resource contains python scripts demostrating how data can be utilised.
